Recursion Tree

                            
Visualizing the permutation generation as a tree structure makes the backtracking process intuitive.

                        

                        
    
                            
  • Root Node: Represents the empty string (start state).
    • 
                            
  • Edges: Represent choosing a character from the available set.
    • 
                            
  • Leaf Nodes: Represent complete permutations (base cases).
    • 
                            
  • Backtracking: Moving up the tree (returning from a child to a parent) restores the available characters.
